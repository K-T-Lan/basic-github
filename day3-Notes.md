## Branch = nhánh làm việc riêng // với nhánh main 
- Làm việc có sai cũng không ảnh hưởng đến main
- Dùng khi: thêm tính năng, sửa lỗi, viết nội dung mới
## All branch -> create new branch
- Quy tắc đặt tên branch: chữ thường, không dấu, dấu / để phân nhóm
## Pull request (PR) = xin gộp code = 'tui làm xong rùi, tui nhập vô main nha!!!"
- Hoàn thiện phần của minh/làm chức năng mới/sửa
